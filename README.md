E-COMMERCE POWER-BI DASHBOARD (AMAZON)

THIS PROJECT ANALYZES E-COMMERCE PRODUCT DATA TO UNDERSTAND PRODUCT PERFORMANCE,PRICING STRATEGIES AND CUSTOMER BEHAVIOUR USING POWER-BI

TOOLS USED : POWER BI , POWER QUERY

KEY FEATURES 

KPI METRICS :TOTAL PRODUCTS,TOTAL REVENUE,AVERAGE RATING
CATOGORY WISE PRODUCT DISTRIBUTION
TOP 10 PRODUCTS BY REVENUE
ACTUAL VS DISCOUNTED PRICE COMPARISION
INTERACTIVE FILTERING USING SLICERS

KEY INSIGHTS

REVENUE IS CONCENTRATED AMONG A FEW TOP PERFORMING PRODUCTS
SIGNIFICANT GAP BETWEEN ACTUAL AND DISCOUNTED PRICES INDICATING STRONG DISCOUNT STRATEGIES
CUSTOMER RATINGS ARE GENERALLY HIGH ACROSS CATOGORIES
